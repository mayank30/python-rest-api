# python-rest-api





##Stuffs we require to build our first REST API
-Python
-Flask
-Flask-SQLAlchemy
-Flask-Restful
-SQlite3
-Jsonify



```
$ virtualenv venv
$ source venv/bin/activate
$ pip install flask flask-jsonpify flask-sqlalchemy flask-restful
$ pip freeze

```

## Run the code 

```
$ python server.py
```
